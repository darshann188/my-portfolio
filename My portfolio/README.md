# Darshan's Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript. Showcases skills, projects, education, and contact information in a clean, professional design.

## 🚀 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Dark Theme**: Professional dark blue gradient background with glassmorphism effects
- **Smooth Animations**: Hover effects, transitions, and animated elements
- **Sections Included**:
  - Hero with introduction
  - About Me with professional summary
  - Education timeline
  - Technical Skills with animated buttons
  - Projects showcase
  - Contact section with social links
- **Accessibility**: Proper ARIA labels, semantic HTML, and keyboard navigation

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and content
- **CSS3**: Modern styling with CSS Grid, Flexbox, and animations
- **JavaScript**: Interactive elements and smooth scrolling
- **Google Fonts**: Inter font family for clean typography

## 🎨 Customization

### Personal Information
- Update name, bio, and contact details in `index.html`
- Replace placeholder text in each section
- Add your real LinkedIn, GitHub, and other social links

### Colors & Styling
- Modify CSS variables in `style.css` under `:root`
- Change `--primary`, `--accent`, and other color variables
- Adjust gradients and backgrounds as needed

### Content
- **Hero Section**: Update the introduction text
- **About Section**: Replace with your professional summary
- **Education**: Add your actual degrees and institutions
- **Skills**: Update programming languages and technologies
- **Projects**: Add real project descriptions and links
- **Contact**: Update email and social media URLs

### Adding Images
- For project screenshots, add image files and update `src` attributes
- Consider adding a profile photo in the About section
